# Sistema de Gestión de Inventarios
## Descripción
Este proyecto implementa un sistema de gestión de inventarios para una tienda. Permite añadir, eliminar, actualizar y buscar productos utilizando programación orientada a objetos en Python.

## Funcionalidades
- Añadir productos
- Eliminar productos por ID
- Actualizar cantidad y precio
- Buscar productos por nombre
- Mostrar todos los productos

## Estructura del Proyecto
- producto.py
- inventario.py
- sistema_inventario.py

## Cómo ejecutar
Ejecutar el archivo sistema_inventario.py desde PyCharm o la terminal.
